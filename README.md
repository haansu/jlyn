# JLyn

JLyn is on it's way to become an image viewer program written in C++ using SFML.

<h3>Supported formats:</h3>
<ul>
	<li>.jpg</li>
	<li>.bmp</li>
	<li>.gif</li>
	<li>.psd</li>
	<li>.hdr</li>
	<li>.pic</li>
	<li>.tga</li>
</ul>
